# Conversor de Números Binarios
Este proyecto permite ingresar un número binario para mostrar
sus conversiones en sistema octal, decimal y hexadecimal. El
proyecto está construido en una aplicación web de Django utilizando
la librería de Bootstrap para el estilo de la página web.

## Instrucciones

Para ejecutar este proyecto, cree un entorno virtual usando
`python -m venv .`

Luego, actívelo invocando el script de activación
`Scripts/Activate`

Instale las dependencias requeridas ejecutando
`pip install -r requirements.txt`

A continuación, aplique las migraciones en Django
`python manage.py migrate`

Por último, corra el servidor de Django con el comando
`python manage.py runserver`
